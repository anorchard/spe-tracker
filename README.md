# SPE Tracker — Semiconductor Equipment Leading Indicator Dashboard

A real-time leading indicator dashboard for semiconductor equipment analysis.

Built with [Perplexity Computer](https://www.perplexity.ai/computer).

## Deployment

This site is hosted on GitHub Pages. To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose the `main` branch and `/ (root)` folder
5. Click **Save**

Your site will be live at `https://<your-username>.github.io/spe-tracker/` within a few minutes.

## Local Development

Simply open `index.html` in a browser, or use any local HTTP server:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```
